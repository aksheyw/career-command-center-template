## ACCUMULATION RULES

After each evaluation or interview prep session:

1. If a new story was improvised → add to `star-stories.md` with full STAR format
2. Update this index with the new story's cross-references
3. If a gap was exposed (no story fit a requirement) → add to GAP TRACKER above
4. After 3+ applications to same company type → review if story selection order should change

**Story quality rule:** Each story must have:

- Specific metrics (not "improved significantly")
- Clear ownership (what YOU did, not the team)
- 60-second tellable version
- Known stumble points
